--------------------------------------------------------------------------------------------------------------------

index.html
    index.html is a hypertext markup language (html) file using the default name of index.html
    html files define the structure of Webpage (the content it displays)
    html files can link to css files to add styling
    html files can link to js files to add functionality

--------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------

styles.css
    styles.css is a cascading style sheets (css) file using the default name of styles.css
    css files define the styling of a webpage (how it looks)
    css files must be linked to an html file using a link tag:

        <link rel="stylesheet" type="text/css" href="<FILEPATH>"> 

        where <FILEPATH> represents the file path to a .css file.

--------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------

script.js 
    script.js is a JavaScript (js) file using the default name of script.js
    js files define the functionality of a webpage (what it does)
    js files must be linked to an html file using a script tag:

        <script src="<FILEPATH>"></script>

        where <FILEPATH> represents the file path to a .js file
